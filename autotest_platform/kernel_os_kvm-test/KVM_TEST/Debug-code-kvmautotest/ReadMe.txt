摘要：本项目是kvm自动化测试平台

项目功能:
在KVM相关的软件包更新后，实现虚拟机自动测试功能

本项目代码库的主要内容就是为实现这个自动化测试平台编写的各种功能和测试脚本

项目中的难点，包括:

项目整体需求的思考
项目整体架构的搭建
项目中关于日志的处理逻辑
项目中有关异常处理的逻辑处理

------------------------------------------------------------------------------
在此之前，常会将调试版作为发布版进行测试，
造成了不必要的麻烦，因此有此目录：kvm测试自动化调试版

本目录文件是为建立本地测试的Debug环境所需
例如，boot.cfg-->boot.cfg.tmp,boardinfo-->/proc/boardinfo
在没有实际机器测试的情况下，需要本地进行测试，
不可以把上面的boot.cfg之类的文件，真正进行修改，
需要在本地进行模拟修改
本目录主要目的是保存相应项目的Debug方法，以备用
